## Panduan Pemakaian

### 1. Clone Repository
Pertama, salin repo ini ke penyimpanan lokal
```bash
git clone https://github.com/Shqcod/StreetObjectClassification-CNN.git
```

### 2. Cek Kesesuaian Python
Pastikan menggunakan versi Python yang kompatibel untuk tensorflow seperti Python 3.10

### 3. Setup Virtual Environemt
Gunakan Virtual Environment untuk menjaga kestabilan dependensi sistem
```bash
# Membuat Virtual Environment
python -m venv venv

# Aktifkan Virtual Environment ( Windows )
source venv/Scripts/activate

# Aktifkan Virtual Environment ( MacOS/Linux )
source venv/bin/activate
```

### 4. Instalasi Library
Pasang semua dependensi yang diperlukan melalui file requirements.txt
```bash
pip install -r requirements.txt
```
